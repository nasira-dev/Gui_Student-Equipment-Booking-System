# Gui_Student-Equipment-Booking-System
Project with colours and in GUI format
# Project Description
This project is developed for Bright Minds Academy. It allows staff to manage equipment bookings using Python.A Python Tkinter-based GUI application for managing student equipment bookings with add, search, view, and cancellation features.
# Features
Add new equipment View equipment list Record booking Prevent double booking View all bookings Search by student Search by equipment Cancel booking View bookings by date Remove equipment
# Data Structures Used
List (for storing equipment and bookings) Dictionary (for storing booking details)
# How to Run
Download Gui_Student-Equipment-Booking-System.py Run using Python 3
